Editing Student Information Request

เราขอแจ้งให้ทราบว่าเรามีการเก็บรวบรวมข้อมูลส่วนบุคคลเพื่อใช้ในการดำเนินการต่าง ๆ ของทางคณะ โดยการให้ความยินยอมในการใช้ข้อมูลของนักศึกษาเป็นสิ่งสำคัญอย่างยิ่ง เราขอให้นักศึกษาเข้าใจและยินยอมในการใช้ข้อมูลของนักศึกษาโดยเต็มที่ หากนักศึกษาไม่ยินยอม หรือมีข้อสงสัยใด ๆ เกี่ยวกับการใช้ข้อมูลจะไม่สามารถเข้าใช้โปรเเกรมในส่วนข้างในได้ หากเกิดข้อสงสัย กรุณาติดต่อเราเพื่อให้เราสามารถให้คำชี้แนะและข้อมูลเพิ่มเติมได้
